Built for the Bazaar.

Shop owners are on WhatsApp 8 hours a day. SUTRA removes the friction of adopting complex ERP software by bringing enterprise logic directly to the chat interface.

Capability Execution
🎙️ Voice-to-Order Hinglish voice notes transcribed via local OpenAI Whisper and mapped to structured database orders in under 30 seconds.
📦 Auto Inventory Stock is deducted automatically upon confirmation. Triggers proactive WhatsApp restocking alerts before a stockout occurs.
💰 Udhaar (Credit) Mgmt Tracks credit limits, ledger aging, and initiates automated, polite payment reminders via WhatsApp.
🧾 GST Invoicing Generates fully compliant PDF invoices and delivers them within the same WhatsApp thread instantly.

Asynchronous
Multi-Agent Mesh.

SUTRA is not a simple chatbot. It is a stateful orchestration engine. No agent calls another directly; all communication routes through Redis Streams for independent scaling and fault tolerance.

01

The Liaison

Ingests the raw chaos of WhatsApp (text or audio). Utilizes Whisper-Hinglish to transcribe and a Sentiment/Intent Analyzer to decode the user's objective.

02

The Strategist

The operational core. Validates the decoded intent against the PostgreSQL database. Checks inventory levels, verifies credit scoring, and calculates pricing.

03

The Auditor

Ensures compliance and immutable record-keeping. Commits the transaction, generates the GST PDF, and dispatches the final response back to WhatsApp.

Infrastructure Stack

  • Backend API FastAPI (Python 3.12+)
  • Database PostgreSQL 15 (Multi-tenant)
  • Event Bus / Queue Redis Streams
  • Speech-to-Text OpenAI Whisper (CPU)
  • Analytics Dashboard Next.js 14 + shadcn/ui
  • Observability Prometheus + Grafana
  • Deployment Docker Compose / systemd
