## Core Architecture

### Agent Mesh Pattern
SUTRA is an **event-driven multi-agent system**. Three specialized agents communicate via Redis pub/sub:

| Agent | Responsibility | Key Integrations |
|-------|---------------|------------------|
| **Liaison** | Intent extraction from WhatsApp text/voice | Whisper-Hinglish, sentiment analysis |
| **Strategist** | Business logic validation & execution | Inventory, credit scoring, pricing |
| **Auditor** | Immutable ledger & compliance | PDF generation, GST validation |

**Critical:** Agents NEVER call each other directly. All communication is via `AgentMessage` objects through Redis Streams. This enables independent scaling and restart.

### AgentMessage Schema (Canonical)
```python
class AgentMessage(BaseModel):
    message_id: str           # UUID, idempotency key
    tenant_id: str            # Isolated per business entity
    source_agent: str         # "liaison" | "strategist" | "auditor"
    payload: dict             # Agent-specific structured output
    confidence: float         # 0.0–1.0 — Liaison populates; others inherit
    requires_confirmation: bool  # True → send WhatsApp confirmation before executing
    timestamp: datetime
```

Each tenant gets:
- Isolated PostgreSQL schema: `tenant_{id}`
- Redis namespace: `sutra:{tenant_id}:*`

## Critical Constraints

### CPU-Only Deployment
- Whisper runs on CPU with acceptable latency for async WhatsApp messaging
- Target deployment: ₹800/month VPS (2 vCPU / 2GB RAM minimum)
- No GPU required in production mode

### Financial Integrity
- **ACID compliance is non-negotiable** for money operations
- Udhaar (credit) entries must be atomic with inventory deductions
- All state changes written to append-only ledger
- Minimum 90% test coverage required for financial calculation modules

### Hinglish NLP Pipeline
Standard Whisper struggles with code-switching. SUTRA adds three post-processing stages:
1. **Language Detection** — segment-level language switching
2. **Domain Vocabulary Injection** — force-decode via `logit_bias`
3. **Normalization** — informal quantities to canonical form (`"teen sau" → 300`)

**Dialect maps location:** `src/nlp/dialect_maps/` — highest-value contribution area
